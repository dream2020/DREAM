Component Description

Functionally, the component allows the therapist to supervise the system by sending commands to the SMS and actuation subsystems. To start a session, the therapist inputs a user name, a session name, and a session partner. Tabs:

  • Load Script: This tab is used start/Stop a script. When the system is automatically following a script, the therapist can approve/disapprove a proposed action. In case a proposed action is disapproved, the therapists can select a new action by clicking off-script buttons. Afterwards, if the button "Back to script" is clicked, the script continues.
  • Between Scripts: This tab is used control the robot when waiting between the scripts. Functions: Dance, Wake up/Sleep, Say a text, enable/disable acttentionReaction subsystem.
  • Child History: This tab is used to display the history of a child.
  • Developer Info: This tab is used to log in/out commands.
