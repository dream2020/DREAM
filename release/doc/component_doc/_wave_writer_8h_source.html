20 // A wave file consists of:
21 //
22 // RIFF header: 8 bytes consisting of the signature "RIFF" followed by a 4 byte file length.
23 // WAVE header: 4 bytes consisting of the signature "WAVE".
24 // fmt header: 4 bytes consisting of the signature "fmt "
25 // fmt size: 4 bytes containing the size of the upcoming WAVEFORMATEX
26 // WAVEFORMATEX: <n> bytes containing a WAVEFORMATEX structure.

28 // wave data: <m> bytes containing wave data.
